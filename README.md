# FinDash — Банковский Dashboard

Личный кабинет в стиле интернет-банкинга.

## Стек
HTML, CSS, JavaScript (без библиотек)

## Фичи
- Карточки баланса, доходов и расходов
- Таблица транзакций с фильтрацией по категории
- Живые курсы валют через ExchangeRate API
- Адаптивная верстка + бургер-меню
